# RangeClarity Agent Instructions

Positioning:
No signals. No hype. Just chart clarity.

Rules:
- Do not promise profits.
- Do not use buy/sell language.
- Do not claim win rates.
- Keep the MVP simple.
- Prefer clean UX over feature overload.
- Do not build AI features yet.
