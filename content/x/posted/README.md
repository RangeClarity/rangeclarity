# content/x/posted/ — audit log (manual)

After a draft is posted live, record it here (and/or update `content/x/status.json`):

- date, the post id (e.g. `POST-1`), the final text as posted, and the X URL.

Nothing here is automated. This folder is the human audit trail referenced by the
approval workflow in `docs/rangeclarity_x_content_agent_plan.md` and
`docs/x-agent-activation.md`.
