# RangeClarity — X Draft Templates

Reusable, on-brand templates. `[brackets]` = fill in. Keep it smart, visual, slightly funny, premium. Never financial advice, buy/sell, targets, or hype. Anonymize chart breakdowns and label them "not a call."

---

## 1. Meme post
```
[One-line clarity-vs-noise joke.]

[Optional second line landing the punch.]

— RangeClarity
```
Image: split or before/after (noisy chart vs one clean level). Minimal, premium.
Example: "Most indicators scream. Ours whispers."

## 2. Build-in-public update
```

```
Pull the specific from `docs/current-sprint.md` / recent patch notes. No metrics/claims.

## 3. Chart structure breakdown

Image: anonymized chart + dashboard. Never name a trade.

## 10. Thread starter
```

1/ Most tools add. We subtract. The signal is in what's NOT on the chart.
2/ Strong structure earns a zone. Weak structure gets a faint line. No structure? We say less.

```
Keep threads short (3–6 posts), one idea each, visual where possible.

---

## Voice cheat-sheet
- Lead with clarity, end with confidence.
- Slightly funny, never goofy; premium, never hype.
- Favorite phrases: "the map, not the move", "clean chart, clear setup", "clarity over noise", "say less", "simple chart, complex engine".
- Always safe: structure / context / levels / bias. Never: buy / sell / target / pump / guaranteed / call.

Invite-only framing only. No fake scarcity, no "limited spots," no outcome promises. At most ~1–2x/week; clarity first, CTA second.
