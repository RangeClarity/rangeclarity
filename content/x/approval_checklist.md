# RangeClarity — X Post Approval Checklist

Run this on **every** draft before posting manually. If any box fails, edit or drop the post. Nothing is auto-posted — a human is always the last step.

## Compliance
- [ ] **No financial advice** — describes structure/context only, never what to do.
- [ ] **No buy/sell wording** — no buy, sell, long, short, entry, exit, target, stop, "defense".
- [ ] **No fake performance claims** — no win rates, "called the top/bottom", returns, backtest brags.
- [ ] **No prediction language** — no "will go up/down", price targets, or probability-of-direction.

## Originality & IP
- [ ] **No copied content** — original wording and memes.
- [ ] **No copyrighted brand logos** or trademarked marks in images.
- [ ] Chart breakdowns are **anonymized** (or clearly educational) and labeled "not a call."

## Brand & quality
- [ ] **No spammy CTA** — no fake urgency, "limited spots", follow-for-follow, mass tagging.
- [ ] **Clear product positioning** — clarity/structure message is obvious.
- [ ] **Readable on mobile** — short lines, scannable, no wall of text.
- [ ] **Visual matches RangeClarity style** — clean, premium, uncluttered; calm tone.
- [ ] Tone is smart / slightly funny / premium — not hype, not goofy.

## Final gate
- [ ] **Human approved** before posting.
- [ ] (Optional) logged in `content/x/posted/` with date + link after posting.

> If unsure whether something reads as advice or hype, cut it. Clarity over noise applies to the marketing too.
