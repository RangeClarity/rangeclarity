# Attribution log

Rule: we build **original** implementations from concepts. We do **not** copy code. If we ever reuse any snippet, it must be from a script whose license explicitly permits reuse, and this log must record the exact source + license + what was reused.

## Policy
1. Concept-level inspiration (an idea like "ATR-clustered S/R") is not copyrightable and needs no permission — but we still log the source we learned it from.
2. **Code reuse** requires: open-source badge confirmed + author's stated reuse terms permit it + attribution per their terms. Mozilla Public License 2.0 is TradingView's default for open-source scripts; respect its notice requirements.
3. Default decision: **reimplement from scratch.** This keeps us clean and avoids the "easy to copy / easy to be accused of copying" trap.
