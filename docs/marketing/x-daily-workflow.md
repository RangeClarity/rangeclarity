# RangeClarity — X Daily Workflow

A 15-minute, repeatable routine. Voice + rules: `x-content-system.md`. Ready posts: `x-post-bank.md`.
**Manual only — nothing here auto-posts, no APIs, no scheduling tools.**

## The 15-minute daily loop
1. **Pick a post type (1 min).** Glance at the weekly rhythm + what you posted yesterday. Don't repeat the same pillar two days running. Lean education-heavy.
2. **Adapt to today's progress (3 min).** Grab a post from `x-post-bank.md` and refresh the specifics — today's chart, today's build note, today's structure idea. Keep the shape, change the detail.
3. **Optional screenshot (3 min).** A clean TradingView chart (ideally with RangeClarity on it) beats text alone for education/product posts. Crop tight, no clutter, no ticker hype. Skip if it would feel forced.
4. **Post on X (1 min).** Under 280 chars. One idea. Re-read against the "never say" list before sending.
5. **Reply to 5 relevant traders (5 min).** Search market-structure / TradingView / "clean chart" conversations. Add one genuinely useful structure observation per reply — no pitch (see template 3).
6. **Log what got engagement (2 min).** One line in a running note: date · post type · rough impressions/replies · what landed. Repeat what works; quietly drop what doesn't.

> Guardrail: if a post starts to sound like a call to buy/sell, predict, or promise an outcome — stop and rewrite it as a *read* of the chart.

## 30-day content themes (4 weeks)
- **Week 1 — Why clarity:** clean-chart philosophy + the cost of noise (pillars 1, 4, 7). Goal: set the tone, build reach.
- **Week 2 — How structure works:** trend, range, location, zones, structure quality (pillars 2, 3). Goal: teach; become the "structure" account people save.
- **Week 3 — Build + product:** build-in-public progress, show the dashboard/screenshots (pillars 5, 6). Goal: credibility + curiosity.
- **Week 4 — Premium + access:** clarity-as-premium positioning, then open beta invites (pillars 1, 8). Goal: convert attention into early beta users.
Repeat the cycle monthly; keep the education : product ratio ≈ 4 : 1.

## Prompt template 1 — generate tomorrow's post
```
Write one X post for RangeClarity (a premium TradingView market-structure clarity tool).
Voice: calm, precise, premium, anti-noise, educational. Positioning: Simple chart, complex engine,
no signals, no noise, just structure — better chart reading, not advice.
Today's pillar: <pillar from the 8>
Today's angle/detail: <one idea, chart, or build note>
Rules: under 280 chars; one idea; NO buy/sell, signals, predictions, profit/win-rate claims, or advice;
no hype/emoji spam; mention RangeClarity only if natural. End with clarity or a question, never a command.
Give me 3 options.
```

## Prompt template 2 — turn a product update into an X post
```
Turn this RangeClarity product update into an X post (build-in-public or product pillar).
Update: <what changed today — feature, refinement, decision, or thing removed>
Frame it as a calm read/clarity improvement, not a signal or outcome. Keep the engine "complex,"
the chart "simple." Under 280 chars, no advice/signals/predictions/profit claims, no hype.
Optionally suggest a one-line screenshot caption. Give me 2 options: one factual, one philosophy-led.
```

## Prompt template 3 — reply to a trader without sounding spammy
```
Write a reply to this trader's post. Be a helpful peer, not a marketer.
Their post: <paste>
Add ONE genuinely useful observation about market structure / chart clarity (e.g., zones vs lines,
location in the range, structure quality). Do NOT pitch RangeClarity, do NOT give buy/sell advice,
do NOT predict price. Calm, specific, under 280 chars. Mention RangeClarity only if they directly ask
what I use — otherwise just be useful.
```

### Weekly review (5 min, Sundays)
Skim your engagement log. Which pillar landed best? Which post format (one-liner vs education vs screenshot)
got saves/replies? Move winners to the top of next week's rotation. Keep the account quiet, useful, and on-brand —
reach follows trust here, not volume.
