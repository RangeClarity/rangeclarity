# sources/

Store downloaded or manually added open-source indicator files here for study — **only** if their license permits storing the code (see `../license_ledger.md`).

## Rules

- One subfolder or file per source, named clearly (e.g. `atr-regime-zeiierman.pine`).
- Never store proprietary, protected, or invite-only scripts. Link to them in the ledger instead.
- Every file here must have a matching row in `../license_ledger.md` and a note in `../notes/`.
- If you may not store the code, record only the URL in the ledger and write the analysis note from observation.

_(This folder is intentionally empty until you add vetted sources.)_
